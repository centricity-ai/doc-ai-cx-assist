## Overview

The AI Chatbot Support Service enables context‑aware customer support using LLM providers, vector search, and a modular backend.

### Capabilities
- Conversational AI with state and context
- Semantic retrieval from knowledge base
- Multi‑provider LLM adapters (Azure OpenAI, AWS Bedrock)
- Production observability and deployment

### Components
- Chat orchestration and sessions
- Knowledge base ingestion and chunking
- Embeddings pipeline and vector storage
- API and integrations
